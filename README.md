This is a [Next.js](https://nextjs.org/) project bootstrapped with TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
